# 🎙️ Empathy Voices
Because voices carry memories words alone can’t.

Empathy Voices lets you type a message you wish you could hear again and plays it back in a tone of your choice — Calm, Reassuring, or Nostalgic.

## 🧠 Built With
- Python  
- Streamlit  
- ElevenLabs API
